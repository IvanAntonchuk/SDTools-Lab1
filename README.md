# Network Simulation Project (Unit Testing Lab)

Цей проєкт симулює роботу мережевого обладнання (Роутери, Комутатори, Графи мережі) та реалізує систему юніт-тестування для перевірки надійності коду.

## Технології
- **Мова:** C++
- **Тестування:** GoogleTest (gtest)
- **Середовище:** Visual Studio 2022

## Результати тестування
Реалізовано **28 юніт-тестів**, що покривають:
1. **Router:** Базова маршрутизація, оновлення таблиць, обробка помилок.
2. **Switch / ManagedSwitch:** Логіка комутації, VLAN, захист портів.
3. **Graph:** Робота з графами (списки/матриці суміжності), перевірка петель.
4. **Network Protocols:** Поліморфізм TCP/UDP пакетів.

## Типи тестів
- **Unit Tests:** Перевірка окремих методів.
- **Edge Cases:** Перевірка граничних умов (від'ємні порти, пусті дані).
- **Behavioral Tests:** Перевірка складної поведінки (Hot Swap, перезапис).
- **Stress Tests:** Навантажувальне тестування (1000 вершин, 500 маршрутів).

## Виправлені помилки (Bug Fixes)
Під час тестування було виявлено критичну помилку в класі `Switch`:
- **Проблема:** Можливість підключення пристрою до неіснуючого порту (наприклад, 100-го).
- **Вирішення:** Додано валідацію номера порту в метод `connectDevice`.
